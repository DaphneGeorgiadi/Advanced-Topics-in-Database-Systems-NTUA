# Advanced Topics in Database Systems – NTUA

This repository contains the implementation of a Big Data analytics project developed for the course Advanced Topics in Database Systems at the National Technical University of Athens (NTUA).

## Overview

The objective of this project is the analysis of large-scale datasets using distributed data processing frameworks. The implementation is based on Apache Spark and Hadoop, and focuses on performance evaluation and optimization techniques in distributed environments.

The project was executed in a cloud environment (AWS) and makes use of both relational and geospatial data processing.

## Datasets

The main dataset consists of crime records in Los Angeles from 2010 to present.

Additional datasets include:

* Police station locations
* Census data (population per area)
* Median household income per ZIP code
* Race and ethnicity codes

All datasets are publicly available and accessed via AWS S3.

## Technologies

* Python (PySpark)
* Apache Spark (RDD, DataFrame, SQL APIs)
* Apache Hadoop (HDFS)
* Apache Sedona (geospatial processing)
* AWS Cloud
